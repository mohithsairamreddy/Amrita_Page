Course Objectives

This course gives an overview of technologies and standards for designing network based applications and solutions. The course provides an insight on the mean stack development using suitable technologies.

  • CO1: Apply the concepts of responsive web design to customize pages for users need. - Understand the principles of responsive web design. - Apply responsive web design techniques to create user-friendly web pages.
  • CO2: Apply markup and scripting languages to design and validate dynamic webpages. - Understand the basics of HTML, CSS, and JavaScript. - Apply these languages to design and validate dynamic webpages.
  • CO3: Evaluate the appropriateness of client/server applications using mean stack architecture. - Understand the different components of the mean stack architecture. - Evaluate the appropriateness of the mean stack architecture for a given client/server application.
  • CO4: Design, develop and deploy client/server applications. - Use the mean stack to design, develop, and deploy client/server applications.
  • CO5: Implement the complete web application development cycle as a term project. - Implement the complete web application development cycle, from requirements gathering to deployment.
